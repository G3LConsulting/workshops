# Workshop rapid prototyping prompts
## Presentation prompts
### Create a task api
**The bad**
```
create a task management API. 
```

**The good**
```
Create a RESTful API for a task management system using .net 9 and c#. The API should support the following functionalities:
1. Create a new task with fields: title, description, due date, creation date, and status (e.g., pending, in-progress, completed).
2. Retrieve a list of all tasks, with optional filtering by status. make sure to support pagination.
3. Update an existing task's details by its ID.
4. Delete a task by its ID.
5. implement OpenAPI documentation for the API endpoints.
Ensure to include proper error handling and use appropriate HTTP status codes for responses. Provide example requests and responses for each endpoint.
```
